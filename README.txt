A2 is a file editor that will Create (C), Delete (D), Open (O), and Read (R) DSU files that are made by the user. A password and username are required to access the files. Using the commands: 

Q -- Quit the program
C -- Create a new file in a specified path
D -- Deleted a file
R -- Read a file
O -- Open a file
E -- Edit a file
P -- Print a file

The program allows users to save username, pwd, and bio into a DSU file and then saves it.
